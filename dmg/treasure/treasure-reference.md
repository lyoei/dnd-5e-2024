# Treasure Reference

> *Source: Dungeon Master's Guide (2024), Chapter 7*

---

## RAW Summary — Currency & Valuables

Standard currency: CP, SP, EP, GP, PP (10:1 ratio each step, EP is 5 SP / 0.5 GP).

## Trade Bars & Goods

Trade bars (various metals, 10–500 GP each) and trade goods (salt, livestock, silk, etc.) serve as alternatives to coins in certain settings.

## RAW Summary — Magic Item Rarity and Value

| Rarity | Value* |
|---|---|
| Common | 100 GP |
| Uncommon | 400 GP |
| Rare | 4,000 GP |
| Very Rare | 40,000 GP |
| Legendary | 200,000 GP |
| Artifact | Priceless |

*Halve the value for a consumable item other than a Spell Scroll. Spell Scroll value = double its scribing cost.

If the item incorporates a base item (e.g., Plate Armor), add that item's cost to the magic item's value.

## RAW Summary — Potion Rules

- **Using a Potion:** Drinking or administering = **Bonus Action**. Effect is immediate. Consumed on use.
- **Mixing Potions:** Roll d100 on the Potion Miscibility table.

### Potion Miscibility Table

| d100 | Result |
|---|---|
| 01 | Both lose effect + magical explosion (5-ft Sphere, 4d10 Force) |
| 02–08 | Both lose effect → ingested poison |
| 09–15 | Both lose effect |
| 16–25 | One potion loses effect |
| 26–35 | Both work, numerical effects/durations halved |
| 36–90 | Both work normally |
| 91–99 | Both work, one potion's numerical effects/duration doubled |
| 00 | Only one works, but effects are **permanent** (DM's choice) |

## RAW Summary — Scroll Rules

- Consumable. Any creature that can read a written language can attempt to use a scroll.
- **Spell Scroll:** Must be on your class's spell list. If spell level > your casting ability, make DC 10 + spell level ability check.

## RAW Summary — Attunement Rules

- Max 3 attuned items at a time.
- Class prerequisites: must be a member of that class.
- "Spellcaster" prerequisite: must be able to cast at least one spell via traits/features (not items).

## RAW Summary — Crafting Magic Items

### Requirements
- **Arcana Proficiency** required for crafter and all assistants
- **Tool Proficiency** in the required tool (see table below)
- If item allows casting spells from it, crafter must have all those spells **prepared every day** of crafting

### Required Tools by Category

| Category | Required Tool |
|---|---|
| Armor | Leatherworker's/Smith's/Weaver's Tools |
| Potion | Alchemist's Supplies or Herbalism Kit |
| Ring | Jeweler's Tools |
| Rod | Woodcarver's Tools |
| Scroll | Calligrapher's Supplies |
| Staff | Woodcarver's Tools |
| Wand | Woodcarver's Tools |
| Weapon | Leatherworker's/Smith's/Woodcarver's Tools |
| Wondrous Item | Tinker's Tools or tool for the base item |

### Crafting Time and Cost

| Rarity | Time* | Cost* |
|---|---|---|
| Common | 5 days | 50 GP |
| Uncommon | 10 days | 200 GP |
| Rare | 50 days | 2,000 GP |
| Very Rare | 125 days | 20,000 GP |
| Legendary | 250 days | 100,000 GP |

*Halved for consumables (except Spell Scrolls — see PHB).

- **Work per Day:** 8 hours. Days needn't be consecutive.
- **Assistants:** Normally 1 assistant max (DM may allow more). Divide time by number of crafters.
- **Raw Materials:** 75% chance available in city, 25% in other settlements. If unavailable, wait 7 days before checking again.
- **Base Item Cost:** If the magic item incorporates a purchasable item (e.g., Plate Armor), also pay that cost or craft it.

## RAW Summary — Artifacts

Artifacts are priceless, unique items. Each typically has:
- **Random Properties** (beneficial + detrimental from tables)
- **Destroying an Artifact** requires a specific method

## RAW Summary — Sentient Magic Items

A sentient item has:
- **Ability Scores** (typically INT, WIS, CHA)
- **Communication** (empathy, speech, or telepathy)
- **Senses** (hearing + sight to a specific range)
- **Alignment and personality**
- **Conflict:** If goals conflict with wielder, contested CHA check. On failure, wielder is Charmed by the item.
