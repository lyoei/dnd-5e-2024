# D&D 2024 vs 2014: Key Rule Changes

This document summarizes the most significant changes from the 2014 PHB to the 2024 revised PHB.

## Terminology Changes

| 2014 Term       | 2024 Term        | Notes                                    |
|-----------------|------------------|------------------------------------------|
| Race            | Species          | Mechanical and narrative redesign        |
| Subrace         | (Removed/Merged) | Traits folded into species or background |
| Racial ASI      | (Removed)        | ASI now comes from Background            |
| Background      | Background (Revised) | Now grants Origin Feat + ASI         |

## Character Creation

### Ability Score Increases
- **2014:** ASI tied to Race (e.g., +2 DEX for Elf)
- **2024:** ASI tied to Background. Choose any +2/+1 or +1/+1/+1

### Backgrounds
- **2014:** Flavor + skill proficiencies + minor feature
- **2024:** Backgrounds are mechanically significant — each grants:
  - 3 Ability Score increases (+2/+1)
  - 2 Skill proficiencies
  - 1 Tool proficiency
  - 1 Origin Feat
  - Equipment package

### Species (formerly Races)
- No longer grant ASI
- Traits simplified and rebalanced
- Half-races removed as separate entries
- Dragonborn: Breath weapon scales, uses = PB/long rest
- Tiefling: Multiple fiendish legacies to choose from

## Classes

### General Changes
- **Subclass levels unified:** Most classes get subclass at Level 3
- **Weapon Mastery:** Martial classes gain Weapon Mastery properties
- **Epic Boons:** Level 19 ASI replaced with Epic Boon feat

### Spellcasting
- **Spell lists reorganized:** Spells now belong to Arcane/Divine/Primal lists, not class-specific
- **Spell preparation:** Many classes that "knew" spells now "prepare" them
- **Cantrip scaling:** Tied to character level, not class level (for multiclass)

### Key Class Changes
- **Ranger:** No longer has favored enemy. Gains Deft Explorer. Major rework
- **Monk:** Ki renamed to Discipline Points. Significant buffs
- **Warlock:** Pact Magic works differently with new Eldritch Invocations
- **Paladin:** Divine Smite is now a spell (bonus action), not a class feature

## Combat

### Action Economy
- **Influence Action (new):** Social interaction in combat formalized
- **Study Action (new):** Learn creature info as an action
- **Two-Weapon Fighting:** No longer uses bonus action for offhand attack (for some classes)

### Weapon Mastery Properties
New system for martial characters:
- **Cleave:** Hit another creature in reach on kill
- **Graze:** Deal ability modifier damage on miss
- **Nick:** Extra offhand attack without bonus action
- **Push:** Push target 10 ft. away
- **Sap:** Disadvantage on target's next attack
- **Slow:** Reduce target speed by 10 ft.
- **Topple:** Target must save or fall prone
- **Vex:** Advantage on next attack against same target

## Feats

### Categories (new)
- **Origin Feats:** Gained from background at Level 1
- **General Feats:** Taken at ASI levels
- **Fighting Style Feats:** Replace the old fighting style class feature
- **Epic Boon Feats:** Level 19+ only

### Notable Changes
- **Great Weapon Master:** No longer -5/+10; instead extra damage = PB on heavy weapon hit
- **Sharpshooter:** No longer -5/+10; reworked
- **Lucky:** Reworked — roll extra d20 and choose
- **Sentinel:** Reworked
- Many feats now have level prerequisites

## Spellcasting

### Spell Lists
- **2014:** Each class had its own spell list
- **2024:** Three master lists: **Arcane**, **Divine**, **Primal**
  - Arcane: Bard, Sorcerer, Warlock, Wizard
  - Divine: Cleric, Paladin
  - Primal: Druid, Ranger

### Other Changes
- Material components: Focus can replace non-consumed, non-costed materials
- Concentration: Rules clarified but largely same
- Upcasting: More spells have meaningful upcast benefits

## Conditions

- **Exhaustion (reworked):** Now a simple -2 per level to d20 rolls, instead of 6 different tiers
- Other conditions largely same but with clarified wording

## Crafting

- **New system** in DMG 2024 for crafting magic items and mundane items
- Requires tools, time, materials, and gold

## Monster Design (MM 2024)

- CR recalculated for many monsters
- Stat blocks reformatted for clarity
- More legendary actions / lair actions for boss monsters
- New monsters added
