## Character Creation

### Backgrounds
- **Ruling:** This table uses **2024 backgrounds** by default.
- **Date:** Session 0
- **Notes:** Players may choose any of the 16 PHB 2024 backgrounds or create a custom one following the background customization rules.

### Legacy Content
- **Ruling:** Legacy 2014 background features require **DM approval**.
- **Date:** Session 0
- **Notes:** If using a 2014 background, ignore its ASIs (use 2024 background ASIs instead). Background features from 2014 should be reviewed for balance against 2024 equivalents.

### Species
- **Ruling:** 2014 species are allowed with the following adjustments: ignore racial ASIs, use background ASIs instead.
- **Date:** Session 0
- **Notes:** See PHB 2024 p.38 sidebar for official guidance on legacy species.
