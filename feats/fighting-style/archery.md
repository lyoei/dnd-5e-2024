# Archery

**Source:** Player's Handbook 2024  
**Category:** Fighting Style Feat  
**Prerequisite:** Fighting Style feature  

## Description

You gain a +2 bonus to attack rolls you make with Ranged weapons.

## 2024 Changes

In the 2014 PHB, Archery was a class feature option (not a feat). The 2024 version converts all Fighting Styles into feats that require the Fighting Style feature as a prerequisite. The mechanical effect (+2 to ranged attack rolls) is unchanged.
