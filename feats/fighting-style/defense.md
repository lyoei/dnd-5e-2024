# Defense

**Source:** Player's Handbook 2024  
**Category:** Fighting Style Feat  
**Prerequisite:** Fighting Style feature  

## Description

While you are wearing Light, Medium, or Heavy armor, you gain a +1 bonus to Armor Class.

## 2024 Changes

In the 2014 PHB, Defense was a class feature option. The 2024 version converts it into a feat. The mechanical effect (+1 AC while wearing armor) is unchanged.
