# Two-Weapon Fighting

**Source:** Player's Handbook 2024  
**Category:** Fighting Style Feat  
**Prerequisite:** Fighting Style feature  

## Description

When you make an extra attack as a result of using a weapon that has the Light property, you can add your ability modifier to the damage of that attack.

## 2024 Changes

In the 2014 PHB, Two-Weapon Fighting was a class feature option that let you add your ability modifier to the damage of the bonus action off-hand attack. The 2024 version converts it into a feat. The core mechanic is similar but now refers to the new Light weapon extra attack system rather than the old bonus action attack.
