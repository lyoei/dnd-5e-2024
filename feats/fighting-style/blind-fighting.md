# Blind Fighting

**Source:** Player's Handbook 2024  
**Category:** Fighting Style Feat  
**Prerequisite:** Fighting Style feature  

## Description

You have Blindsight with a range of 10 feet.

## 2024 Changes

In the 2014 PHB, Blind Fighting was introduced in Tasha's Cauldron of Everything as a Fighting Style class feature option. The 2024 version converts it into a feat. The mechanical effect (Blindsight 10 ft.) is unchanged.
