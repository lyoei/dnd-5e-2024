# Modify Memory

*5th-level Enchantment*

- **Casting Time:** Action
- **Range:** 30 feet
- **Components:** V, S
- **Duration:** Concentration, up to 1 minute
- **Concentration:** Yes
- **Ritual:** No

## Description

You attempt to reshape another creature's memories. One creature that you can see must make a Wisdom saving throw. If you are fighting the creature, it has Advantage on the saving throw. On a failed save, the target has the Charmed condition for the duration. While Charmed, the target is Incapacitated and unaware of its surroundings. You can affect the target's memory of an event that it experienced within the last 24 hours and that lasted no more than 10 minutes. You can permanently eliminate all memory of the event, allow the target to recall the event with perfect clarity, change the target's memory of the details of the event, or create a memory of some other event.

## At Higher Levels

When you cast this spell using a spell slot of 6th level or higher, you can alter memories from further back: 6th (7 days), 7th (30 days), 8th (1 year), 9th (any time).

Modernized with Charmed and Incapacitated conditions.
