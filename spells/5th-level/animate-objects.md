# Animate Objects

*5th-level Transmutation*

- **Casting Time:** Action
- **Range:** 120 feet
- **Components:** V, S
- **Duration:** Concentration, up to 1 minute
- **Concentration:** Yes
- **Ritual:** No

## Description

You animate up to ten nonmagical objects within range. Medium or smaller objects count as one; Large objects count as two; Huge objects count as four. Each animated object uses the Animated Object stat block, with its HP, AC, and attack determined by its size. On your turn, you can mentally command the objects (no action required). If you command multiple objects, they all do the same thing. Objects act on your turn.

## At Higher Levels

When you cast this spell using a spell slot of 6th level or higher, you can animate two additional objects for each slot level above 5th.
