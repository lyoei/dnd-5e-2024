# Guidance

*Cantrip, Divination*

- **Casting Time:** Action
- **Range:** Touch
- **Components:** V, S
- **Duration:** Concentration, up to 1 minute
- **Concentration:** Yes
- **Ritual:** No

## Description

You touch a willing creature and choose a skill. Until the spell ends, the creature adds 1d4 to any ability check using the chosen skill.

## Spell Lists

Cleric, Druid

## 2024 Changes

Now requires choosing a specific skill when cast, and applies to all checks using that skill for the duration, rather than being a one-time use d4 added to any ability check. Slightly more focused but repeatable within the duration.
