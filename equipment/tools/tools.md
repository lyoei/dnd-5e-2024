# Tools

A tool helps you make specialized ability checks, craft certain items, or both. Each tool's description includes its cost, weight, and the following entries:

- **Ability:** The ability to use when making an ability check with the tool.
- **Utilize:** Things you can do with the tool when you take the Utilize action (with DC).
- **Craft:** What you can craft with the tool (using the crafting rules).
- **Variants:** If the tool has variants, each requires a separate proficiency.

## Tool Proficiency

If you have proficiency with a tool, add your **Proficiency Bonus** to any ability check you make that uses the tool. If you also have proficiency in a **skill** used with that check, you have **Advantage** on the check.

### Notable Other Tools

- **Thieves' Tools:** Essential for Rogues. Used for picking locks and disarming traps. Required for many dungeon exploration checks.
- **Herbalism Kit:** Used to craft Antitoxin and Potions of Healing (per the crafting rules).
- **Poisoner's Kit:** Used to craft poisons. Includes vials, chemicals, and a glass stirring rod.
- **Disguise Kit:** Contains cosmetics, hair dye, small props, and clothing. Used for impersonation.
- **Forgery Kit:** Includes papers, inks, seals, and wax. Used to create convincing forgeries.
- **Navigator's Tools:** Includes a sextant, compass, calipers, ruler, parchment, ink, and quill. Used for plotting courses.

## 2024 Changes

Key changes from the 2014 PHB:

- **Ability scores assigned to each tool (NEW):** In 2014, tools didn't have a fixed ability score — the DM decided which ability applied. The 2024 PHB assigns a specific ability to each tool (e.g., Smith's Tools → Strength, Thieves' Tools → Dexterity).
- **Utilize action integration (NEW):** Each tool now has a specific "Utilize" entry describing what you can do with the Utilize action, including a DC. This replaces the vague 2014 guidance.
- **Craft entry (NEW):** Each tool explicitly lists what it can craft, tying into the new crafting system.
- **Skill + Tool double proficiency → Advantage (NEW):** If you have proficiency in both the tool AND a relevant skill, you gain Advantage on the check. In 2014 (via Xanathar's Guide), the rule was to add double proficiency bonus instead.
- **Gaming Sets use Wisdom:** Previously the ability for gaming sets was unspecified; now explicitly Wisdom.
- **Musical Instruments use Charisma:** Previously unspecified; now explicitly Charisma for all instruments.
