## 2024 Changes

Key changes from the 2014 PHB:

- **Mechanical rules for many items (EXPANDED):** Items like Ball Bearings, Caltrops, Acid, Alchemist's Fire, and Holy Water now have explicit save DCs using your own DEX modifier + Proficiency Bonus (previously flat DCs or DM discretion).
- **Utilize action integration (NEW):** Many items now specify they use the "Utilize action" — a new action type in 2024 that covers non-combat tool and item use.
- **New items added:** Book, Bright Fungal Cloak, Costume, Desert Clothing, Devil Mask, Garb of Light and Shadow, Genie Robe, Locking Spellbook, Monster Camouflage — all new to the 2024 PHB.
- **Component Pouch and Focuses:** Functionally similar to 2014, but Ranger now explicitly uses Druidic Focuses.
- **Equipment Packs:** Largely similar to 2014 but adjusted for the revised item list.
- **Antitoxin:** Now explicitly a Bonus Action to drink (previously unspecified action economy).
